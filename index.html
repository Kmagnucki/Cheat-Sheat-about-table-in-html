Ściągawka z Tabel html.

Table Tags

Tag Nazwa Wyjaśnienie
<table> Table Opakowanie elementow tabeli.
<thead> Table Head Zestaw wierszy definiujących nagłówki kolumn w tabeli.
<tbody> Table Body Zestaw wierszy zawierających rzeczywiste dane tabeli.
<tr> Table Row Komórka wiersza tabeli(W której znajdują sie <tr>).
<td> Table Data Kamórka z rzeczywistą informacją w tabeli.
<th> Table Header Komórka z rzeczywistą informacją w tabeli NAGŁÓWKU.
<tfoot> Table Foot Zestaw wierszy definiujących stopkę w tabeli.

Table Atrybuty

Atrybut Nazwa Wyjaśnienie
colspan Column Span Definiuje ile KOLUMN powinien obejmować element td.
rowspan Row Span Określa, ile WIERSZY powinien obejmować element td.
